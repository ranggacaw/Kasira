## ADDED Requirements
### Requirement: Premium Reporting Workspace Presentation
Kasira SHALL present entitled reporting workflows with a premium multi-panel workspace that emphasizes export actions, profitability highlights, payment reporting, cashier insights, and product or inventory summaries.

#### Scenario: Entitled user opens reports
- **WHEN** an entitled user opens a reporting workflow
- **THEN** the page organizes its reporting modules and export actions using the shared premium information hierarchy instead of a generic form-and-list presentation
